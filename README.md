## hackathon 黑客马拉松

### 项目背景

##### 专门为了黑客马拉松做的一个项目,项目前端使用reactjs做基础架构。

### 技术方案

* 规范：
    * ES6
    * CommonJS
* 基础库：
    * Reactjs
    * Reflux
    * ReactRouter
    * Zepto
    * Q
* 样式：
    * Stylus
* 调试及构建：
    * JSX
    * Sourcemap
    * browserSync
    * Webpack
    * Gulp
